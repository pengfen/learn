什么是 Java 中的字符串 在程序开发中字符串无处不在 如用户登录时输入的用户名 密码等使用的就是字符串 在 Java 中 字符串被作为 String 类型的对象处理 String 类位于 java.lang 包中 默认情况下 该包被自动导入所有的程序 创建 String 对象的方法 String s1 = "peng"; // 创建一个字符串对象 peng 名为 s1 String s2 = new String(); // 创建一个字符串对象 名为 s2 String s3 = new String("peng"); // 创建一个字符串对象 peng 名为 s3 String 对象创建后则不能被修改 是不可变的 所谓的修改其实是创建了新的对象 所指向的内存空间不同 String s1 = "peng"; String s2 = "peng"; String s3 = new String("peng"); String s4 = new String("peng");
