Java 中数组 数组 是一组相关数据的集合 数组中的数据类型一定是完全相同 数组的三种类型 一维数组 二维数组 多维数组 要使用数组 首先要声明数组 然后要分配具体的内容给数组 int scores[] = null; // 声明 scores 为整型数组 scores = new int[10]; // 为整型数据分配空间 其元素个数是 10 for (int x = 0; x < 10; x ++) { // 打印出数组中每一个元素的值 System.out.println(scores[x]); } 执行结果如下 在内存的栈中保存数组的名字 而在内存的堆中则保存数组的具体内容 数组的名字会具体指向数组的内容 从而可能过数组的名字加上数组内容索引的方式来访问数组中的具体内容 数组中有一些类同类型的元素 每个元素的访问可以通过数组的名称加上元素的索引位置来共同完成 数组的索引位置是从 0 开始的 例如访问数组中的第三个元素 scores[2]
